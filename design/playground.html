请用 Python 写一个快速排序，并加上中文注释
Assistant · GPT-4o
当然可以，这里是一个带中文注释的快速排序实现：
def quick_sort(arr):
    # 基线条件：如果数组长度小于等于1，直接返回
    if len(arr) <= 1:
        return arr
    # 选择基准元素（这里取中间值）
    pivot = arr[len(arr) // 2]
    # 小于基准的元素放左边
    left = [x for x in arr if x < pivot]
    # 等于基准的元素放中间
    middle = [x for x in arr if x == pivot]
    # 大于基准的元素放右边
    right = [x for x in arr if x > pivot]
    # 递归排序左右两部分并合并
    return quick_sort(left) + middle + quick_sort(right)

时间复杂度：
• 平均情况：O(n log n) — 每次划分大致平衡
• 最坏情况：O(n²) — 基准元素总是最大/最小值

空间复杂度：
• 上述实现为 O(n)（列表推导创建新数组）
• 原地分区版本可优化至 O(log n) 栈空间
