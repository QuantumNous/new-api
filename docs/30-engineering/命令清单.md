# 命令清单

## 文档命令
```bash
task docs:check
task ai:check
task docs:new-adr -- title=某个架构决策
```

## 后端命令

```bash
./start.sh
go run . --port 3000
go test ./...
go test ./model ./controller ./router
```

## 前端命令
默认前端位于 `web/default/`，优先使用 Bun：

```bash
cd web/default
bun install
bun run dev
bun run i18n:sync
bun run typecheck
bun run lint
bun run format:check
bun run build:check
```
