# 页面清单

## 公共入口

- `/`：默认 Lighting 首页；支持系统配置的 URL、HTML、Markdown 自定义首页。
- 登录、注册、忘记密码、OTP、重置、OAuth 回调：统一品牌卡片布局。

## 组织用户侧

- `/organization/usage`：组织汇总、趋势、模型和渠道用量。
- `/organization/members`：Owner/Admin 成员与组织名称管理。
- `/organization/logs`：Owner/Admin/Billing 消费明细与导出。

## 组织管理员侧

- `/admin/organizations`：组织搜索、状态筛选、创建和分页。
- `/admin/organizations/:id`：Members、Billing、Logs 三个 Tab。

组织页面的详细状态与角色可见性见 [组织管理与组织账单 UI 设计](./组织管理与组织账单UI设计.md)。品牌和默认首页见 [默认前端品牌与视觉设计](./默认前端品牌与视觉设计.md)。
