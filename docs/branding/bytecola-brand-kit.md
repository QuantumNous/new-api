# ByteCola Brand Kit

ByteCola 的品牌目标是：让 AI 像可乐一样即开即用。

## Brand Core

- Brand Name: `ByteCola`
- Chinese Name: `字节可乐`
- Primary Slogan: `让 AI 像可乐一样即开即用`
- English Slogan: `AI, open and ready like cola.`
- Brand Personality: `即时、轻快、亲和、可靠、可运营`

## Core Assets

以下颜色基于当前主 Logo 反推并固化为项目品牌标准值，后续前端、运营物料与后台品牌配置统一使用这些值。

| Asset | Value | Usage |
| --- | --- | --- |
| Byte Blue | `#1494D1` | 主品牌色，Logo 蓝色部分、主按钮、重点链接 |
| Cola Red | `#DB0D18` | 辅品牌色，强调按钮、徽标点缀、重点标签 |
| Brand Background | `#F7F4F1` | 品牌浅底、落地页背景、大面积留白区域 |
| Bubble Mist | `#D9E2E7` | 辅助灰，用于气泡、分隔、弱提示背景 |
| Deep Ink | `#11212B` | 深色正文、深色标题、深背景上的替代主文字色 |

## Typography

- Logo Typeface: `Fredoka Bold`
- Recommended Chinese UI Typeface: `Noto Sans SC`
- Recommended Latin UI Typeface: `Inter`

说明：

- `Fredoka Bold` 用作当前 ByteCola 品牌字标的指定字体，字形圆润、开口友好，和现有 Logo 气质最接近。
- 正文字体不与 Logo 字体混用；产品正文与控制台界面优先使用 `Noto Sans SC, Inter, system-ui, sans-serif`。

## Logo System

项目当前补齐的 Logo 套件如下：

| File | Purpose |

## Logo Usage Rules

- `64px` 以下严禁使用整版字标，统一使用从原图提取的 `Logo Mark`。
- 登录页、顶部导航、页脚与站点分享图优先使用 `logo.png`。
- 浏览器 favicon 使用 `Logo Mark`，移动端图标优先使用 `App Icon`。
- Logo 周围至少保留 `0.25x` 安全留白，`x` 为 Logo 图形高度。
- 禁止拉伸、描边、改色、添加投影、旋转或叠加复杂背景纹理。

## Brand Voice

- 对外表达要短、准、直接。
- 核心卖点突出“即开即用”“统一入口”“快速接入”“稳定交付”。
- 避免空泛技术口号，优先说明用户能马上得到什么。
