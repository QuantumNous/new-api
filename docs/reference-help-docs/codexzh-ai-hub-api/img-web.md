# img.xbai.top 在线图片生成教程

无需安装任何客户端，打开 [img.xbai.top](https://img.xbai.top/) 即可调用 AI HUB API 生成 AI 图像。

* * *

> **前置条件**
>
> -   已有 AI HUB API 令牌（没有？查看 [快速开始](https://docs.codexzh.com/ai-hub-api/quick-start)）
> -   创建令牌时分组选择 **默认分组（default）**

## 第一步：打开在线生图网站

访问 **[https://img.xbai.top](https://img.xbai.top/)**，无需注册登录，直接进入主界面。

![img.xbai.top 主界面](https://docs.codexzh.com/assets/11.BEdbjXEy.png)

## 第二步：填写 API Key

点击右上角**设置**图标，在弹窗中将 API Key 填入 **apikey** 输入框，然后关闭弹窗。

![打开设置，填写 API Key](https://docs.codexzh.com/assets/12.CXX0T2Zt.png)

> API Key 前往 [api.xbai.top](https://api.xbai.top/) 创建令牌获取，分组选择**默认分组（default）**。详细步骤参考 [快速开始](https://docs.codexzh.com/ai-hub-api/quick-start)。

## 第三步：选择模型并生成图片

填写 API Key 后直接输入提示词即可生成图片。如需更换模型，在**模型 ID** 输入框中填写对应模型名称。

![填写模型 ID 并生成图片](https://docs.codexzh.com/assets/13.Bs2OcrR2.png)

![生成效果示例](https://docs.codexzh.com/assets/14.R_7Kh2I8.png)

> 可用生图模型列表：前往 [api.xbai.top/pricing](https://api.xbai.top/pricing)，在「计费类型」中选择**按次计费**查看。

* * *

## 遇到问题？

-   生成报错：检查 API Key 是否正确，令牌分组是否选择了「默认分组」
-   找不到想用的模型：前往 [定价页面](https://api.xbai.top/pricing) 查看按次计费模型列表
-   联系客服：[客服支持](https://docs.codexzh.com/ai-hub-api/support)
