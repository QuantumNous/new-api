# AI HUB API 中转

官方网址

**AI Hub 官方网址：** [https://api.xbai.top](https://api.xbai.top/)

AI HUB API 提供稳定的 AI 模型访问通道，支持 Claude、GPT、Gemini 等主流模型。

## 快速上手

> 新手推荐流程：
>
> 1.  注册账号并购买额度
> 2.  创建 API 令牌（选择对应分组）
> 3.  使用 CC-Switch 一键配置（推荐）

详细教程见 [快速开始](https://docs.codexzh.com/ai-hub-api/quick-start)。

## 支持的工具

### 命令行工具

-   **Claude Code**：官方 AI 编程助手
-   **Codex**：OpenAI 代码编辑器
-   **Gemini CLI**：Google AI 命令行工具

### IDE 插件

-   **VS Code**：Claude Code、Cline、Roo Code
-   **Cursor**：原生集成
-   **其他编辑器**：支持 OpenAI 兼容格式的工具

## 服务地址

-   API 地址：`https://api.xbai.top`
-   控制台：`https://api.xbai.top`
