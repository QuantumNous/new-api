## 如何查看最新分组

1.  登录控制台，点击右上角「模型广场」，左侧显示所有可用分组，右侧显示该分组下的模型列表

![image-20260201185900358](https://docs.codexzh.com/assets/4.BVWeXb-g.jpg)

2.  分组名后的倍率（如 x0.5、x1.0）表示计费倍率：

-   倍率 < 1：享受折扣（0.5 = 5折，0.8 = 8折）
-   倍率 = 1：原价计费
-   倍率 > 1：需加价（1.5 = 额外支付 50%）

> 通过模型广场可以看到每个分组支持的所有模型，避免配置后提示「模型不存在」。

## 默认分组

> 在自己的程序中或第三方客户端中使用，选择默认分组即可

-   分组介绍：模型
-   支持工具：第三方插件、第三方客户端、API 接口请求等
-   是否支持第三方：✅ 支持
-   计费倍率：x1
-   模型列表：[点击查看实时模型列表](https://api.xbai.top/pricing)

### CC 分组

> 使用 Claude Code 必选此分组

-   分组介绍：专为 Claude Code 优化，提供最稳定的 Claude 模型访问
-   支持工具：Claude Code CLI、VS Code Claude Code 插件
-   是否支持第三方：❌ 不支持（为保障账号安全，禁止接入第三方工具）
-   计费倍率：x1.88
-   模型列表：[点击查看实时模型列表](https://api.xbai.top/pricing)

使用建议：

-   ✅ 推荐用于 Claude Code 编程开发
-   ❌ 不要接入其他第三方工具，避免账号封禁

### Codex 分组

> 使用 Codex 必选此分组

-   分组介绍：专为 OpenAI Codex 编辑器优化，提供稳定的代码生成能力
-   支持工具：Codex CLI、Cursor、Cline
-   是否支持第三方：✅ 支持
-   计费倍率：x0.5
-   模型列表：[点击查看实时模型列表](https://api.xbai.top/pricing)

使用建议：

-   ✅ 推荐用于 Codex、Cursor 等代码编辑器
-   ✅ 可接入支持 OpenAI 格式的第三方工具

## 选择建议

| 使用工具 | 推荐分组 |
| --- | --- |
| Claude Code | CC 分组 |
| Codex / opencode | Codex 分组 |

### 常见错误

使用 Claude Code 但选了其他分组：提示「模型不存在」或连接失败，必须选择 CC 分组。

使用 Codex 但选了 CC 分组：无法访问 Codex 专用模型，切换到 Codex 分组。

不确定分组下有哪些模型：访问模型广场查看实时模型列表。

## 动态更新

分组和模型会不定期更新，请以控制台「模型广场」的实时信息为准。
