# 分组定价

了解各分组的计费方式和价格。

## 计费说明

### 基本规则

-   计费单位：按 Token 使用量计费（1K tokens = 1000 个 token）
-   计费方式：预付费，从账户余额扣除
-   充值比例：1:1（1 元人民币 = 1 美元额度）
-   额度有效期：永久有效，不过期

* * *

### 计费倍率

不同分组有不同的计费倍率，影响最终价格：

| 倍率 | 说明 | 示例 |
| --- | --- | --- |
| x0.5 | 5 折优惠 | 原价 $0.10，实付 $0.05 |
| x1.0 | 原价计费 | 原价 $0.10，实付 $0.10 |
| x1.5 | 加价 50% | 原价 $0.10，实付 $0.15 |

> 查看倍率：登录控制台，访问「模型广场」，分组名后会显示倍率。

## 常见问题

**如何查看当前余额？**

登录控制台，右上角显示当前余额。

**余额不足会怎样？**

API 调用会失败，提示「余额不足」。不会产生欠费，仅停止服务。充值后立即恢复。

**可以退款吗？**

未使用的充值额度可申请退款，手续费 5%，已使用部分不支持退款。

**价格会变动吗？**

分组倍率相对稳定，模型基准价格跟随官方调整，重大变动会提前公告。
