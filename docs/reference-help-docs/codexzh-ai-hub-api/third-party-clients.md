## DeepChat

第三方跨平台桌面客户端，支持安装包与包管理器两种安装方式。

-   类型：第三方桌面客户端（跨平台）
-   支持方式：官方安装包 + 包管理器

### 官方链接

-   [官方下载页](https://deepchat.thinkinai.xyz/#/download)
-   [安装指南](https://deepchatapp.github.io/documents/guide/getting-started/installation.html)
-   [GitHub 仓库](https://github.com/ThinkInAIXYZ/deepchat)

* * *

### Windows 版本

#### 安装包 / 便携版

前往 [官方下载页](https://deepchat.thinkinai.xyz/#/download) 选择版本下载

#### 包管理器安装（推荐）

bash

```
# 安装
winget install DeepChat

# 更新
winget upgrade DeepChat
```

* * *

### macOS 版本

#### DMG 安装包

前往 [官方下载页](https://deepchat.thinkinai.xyz/#/download) 选择版本下载

#### Homebrew 安装（推荐）

bash

```
# 安装
brew install --cask deepchat

# 更新
brew upgrade --cask deepchat
```

* * *

### Linux 版本

#### Ubuntu / Debian (APT)

bash

```
# 添加 GPG 密钥
curl -s https://deepchat.thinkinai.xyz/linux/ubuntu/gpg.key | sudo apt-key add -

# 添加 DeepChat 仓库
sudo add-apt-repository "deb [arch=amd64] https://deepchat.thinkinai.xyz/linux/ubuntu stable main"

# 更新软件包列表
sudo apt update

# 安装 DeepChat
sudo apt install deepchat
```

#### Fedora / RHEL (DNF)

bash

```
# 导入 GPG 密钥
sudo rpm --import https://deepchat.thinkinai.xyz/linux/fedora/gpg.key

# 添加 DeepChat 仓库
sudo dnf config-manager --add-repo https://deepchat.thinkinai.xyz/linux/fedora/deepchat.repo

# 安装 DeepChat
sudo dnf install deepchat
```

#### Arch Linux (AUR)

bash

```
# 使用 yay
yay -S deepchat

# 或使用 paru
paru -S deepchat
```

#### 通用 Linux (AppImage)

前往 [官方下载页](https://deepchat.thinkinai.xyz/#/download) 下载 AppImage 版本

bash

```
# 下载后运行
chmod +x DeepChat-*.AppImage
./DeepChat-*.AppImage
```

## 常见问题

**Q: 这些客户端安全吗？**

均为知名开源项目。建议从官方 GitHub 或官方网站下载，关注项目星标和社区评价，并定期更新。

**Q: 下载太慢怎么办？**

Cherry Studio 提供多个镜像源，切换不同线路即可。

**Q: 可以离线使用吗？**

不可以。这些客户端需要联网才能调用 AI 模型服务。

**Q: 我的数据安全吗？**

对话数据存储在所选 AI 服务商的服务器上（如 OpenAI、Anthropic 等），请参考各服务商的隐私政策。
