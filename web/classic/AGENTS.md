# web/classic/ — Classic Frontend

## Overview
Legacy frontend theme. React 18 + JS (JSX) + Semi Design + react-router-dom v6.

## Tech Stack
- React 18 (not 19)
- JavaScript (JSX), no TypeScript
- Semi Design component library
- react-router-dom v6
- React Context for state management
- Manual Axios (no React Query / Zustand / Zod)

## Conventions
- ESLint 8 with AGPL-3.0 header enforcement.
- Tailwind v3 (if used).

## Anti-Patterns
- Do NOT add new features here unless explicitly required. Default to `web/default/`.
