# 登录使用

登录你的 61kj 账号。

## 登录方式

当前仅支持账号密码登录，请使用注册时填写的用户名和密码进行登录。

1. **进入登录页面**
打开 61kj 登录页，进入账号登录界面。

2. **输入账号密码**
填写注册时使用的用户名和密码。

3. **完成登录**
提交后即可进入控制台，继续进行充值、创建令牌和接口调用。
