# 1. 进入项目根目录

cd /Users/liangyueyouqi/Desktop/算力中转/new-api

# 2. 启动后端 + PostgreSQL + Redis

docker compose -f docker-compose.dev.yml up -d --build

# 3. 检查后端是否启动成功

curl http://localhost:3000/api/status

# 4. 进入默认前端目录

cd web/default

# 5. 安装前端依赖

bun install

# 6. 启动默认前端开发服务

bun run dev

启动后打开终端里显示的前端地址，一般是：

http://localhost:3001

停止后端开发环境：
cd /Users/liangyueyouqi/Desktop/算力中转/new-api
docker compose -f docker-compose.dev.yml down

如果你要启动经典前端，用另一个终端执行：
cd /Users/liangyueyouqi/Desktop/算力中转/new-api/web/classic
bun install
bun run dev
